# 👨‍💻 Currículo Online | Portfólio Pessoal

Este projeto consiste em um currículo online desenvolvido com foco em organização, clareza e boa experiência do usuário.  
Foi criado para apresentar minhas competências técnicas e informações profissionais de forma moderna e responsiva.

---

## 🎯 Objetivo

Desenvolver uma aplicação web simples, estruturada e visualmente agradável para demonstrar meus conhecimentos em desenvolvimento front-end.

---

## 🚀 Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (estilização e responsividade)
- JavaScript (interatividade)

---

## 📋 Funcionalidades

- ✅ Layout responsivo
- ✅ Estrutura organizada e de fácil leitura
- ✅ Seção de habilidades técnicas
- ✅ Seção de experiências e formação
- ✅ Informações de contato
